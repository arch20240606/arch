РОЛИ
1) ROLE_ADMIN_EXPERTISE - Администратор справочников в модуле Экспертизы
2) ROLE_ADMIN_SECURITY - Администратор информационной безопасности
3) ROLE_ADMIN_USER - Администратор для создания, редактирования пользователей
4) ROLE_GO_EXPERTISE_CONFIRMER - Пользователь ГО с возможностью согласования сведений по экспертизе НТД в сфере информатизации (Должна быть проставлена организация)
5) ROLE_GO_EXPERTISE_EDITOR - Пользователь ГО исполнитель с возможностью редактирования сведений по экспертизе НТД в сфере информатизации (Должна быть проставлена организация)
6) ROLE_GO_EXPERTISE_HEAD - Пользователь ГО руководитель структурного подразделения с возможностью отправки НТД для экспертизы в УО в сфере информатизации (Должна быть проставлена организация)
7) ROLE_GO_EXPERTISE_SIGNER - Пользователь ГО с возможностью подписывания для дальнейшей отправки руководителем НТД для экспертизы (Должна быть проставлена организация)
8) ROLE_SI_EXPERTISE_CONFIRMER - Пользователь СИ с возможностью согласования заключения по экспертизе НТД в сфере информатизации (Не должна быть проставлена организация)
9) ROLE_SI_EXPERTISE_HEAD - Пользователь СИ руководитель структурного подразделения экспертизы НТД в сфере информатизации (Не должна быть проставлена организация)
10) ROLE_SI_EXPERTISE_REVIEWER - Пользователь СИ исполнитель с возможностью экспертизы НТД в сфере информатизации
11) ROLE_SI_EXPERTISE_SIGNER - Пользователь СИ с возможностью подписывания заключения экспертизы НТД (Не должна быть проставлена организация)
12) ROLE_STS_EXPERTISE_CONFIRMER - Пользователь ГТС с возможностью согласования заключения по экспертизе НТД в сфере информатизации (Не должна быть проставлена организация)
13) ROLE_STS_EXPERTISE_HEAD - Пользователь ГТС руководитель структурного подразделения экспертизы НТД в сфере информатизации (Не должна быть проставлена организация)
14) ROLE_STS_EXPERTISE_REVIEWER - Пользователь ГТС исполнитель с возможностью экспертизы НТД в сфере информатизации
15) ROLE_STS_EXPERTISE_SIGNER - Пользователь ГТС с возможностью подписывания заключения экспертизы НТД (Не должна быть проставлена организация)
16) ROLE_UO_EXPERTISE_CONFIRMER - Пользователь УО с возможностью согласования заключения по экспертизе НТД в сфере информатизации (Не должна быть проставлена организация)
17) ROLE_UO_EXPERTISE_HEAD - Пользователь УО руководитель структурного подразделения экспертизы НТД в сфере информатизации (Не должна быть проставлена организация)
18) ROLE_UO_EXPERTISE_REVIEWER - Пользователь УО исполнитель с возможностью экспертизы НТД в сфере информатизации
19) ROLE_UO_EXPERTISE_SIGNER - Пользователь УО с возможностью подписывания заключения экспертизы НТД (Не должна быть проставлена организация)
